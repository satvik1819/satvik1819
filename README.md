## 🧑‍💻 &nbsp; Who Am I?

I'm **Lakshmi Narasimha Naga Sathvik Vedantham**, a final-year B.Tech CSE student specialising in Data Science & Data Engineering at Lovely Professional University, Punjab, currently gaining hands-on industry experience as a **Data Science Intern at Pickl.ai**.

My work sits at the intersection of **Generative AI**, retrieval-augmented generation, and scalable backend engineering. I care deeply about building systems that are not just accurate, but *explainable, grounded, and shippable* — systems that think, not just execute. Right now I'm channelling that into an AI-powered shopping assistant, advanced RAG pipelines with semantic chunking, and multi-agent workflows built around memory and reasoning.

I'm open to roles in **AI/ML Engineering**, **Data Science**, and **Backend Engineering**. Outside of building, you'll find me competing in hackathons — most recently earning **Runner-up at the Cederstrom Hackathon** against 100+ teams in a 24-hour sprint — earning cloud certifications, and pushing the boundaries of what autonomous AI systems can do.

## 🚀 &nbsp; What I'm Building Right Now

<br/>

**🛒 AI-Powered Shopping Assistant** — A conversational agent with multi-turn memory, hallucination-free responses via RAG, hybrid product recommendations (content-based + collaborative), and LLM-powered review summarization. Built with LangChain, vector databases, and FastAPI on the backend.

**🔬 Advanced RAG Pipelines** — Experimenting with structured retrieval strategies, semantic + keyword hybrid search, re-ranking, and context compression to push retrieval accuracy beyond naive top-k.

**🤖 Multi-Agent AI Systems** — Designing agents with persistent memory, dynamic planning loops, and tool-use capabilities. Goal: systems that can *reason across steps*, not just respond to prompts.

### 📡 &nbsp; Multivariate Time Series Anomaly Detection &nbsp;`Jan 2026`

> **Python · Scikit-learn · TensorFlow · LSTM**

An anomaly detection system for industrial equipment sensor data, framed as a rigorous head-to-head comparison between **Isolation Forest** (classical ML) and **LSTM networks** (deep learning).

The pipeline handles raw multivariate sensor signals — sliding window segmentation, normalization, and feature engineering — feeding into both models simultaneously. Visualizations reveal where each approach shines and where it fails, giving practitioners an honest, reproducible benchmark for sensor anomaly workloads.

### 🛒 &nbsp; Product Return Prediction Model &nbsp;`Sep 2025`

> **Python · Scikit-learn · Predictive Modeling**

An end-to-end ML classification pipeline that predicts e-commerce product returns with **89% accuracy** — turning a costly business problem into a tractable machine learning one.

The pipeline spans data cleaning, feature engineering, class imbalance handling (resampling strategies), model training, and evaluation via confusion matrix and classification report. The deliverable isn't just a model — it's actionable business recommendations extracted directly from prediction outputs, ready to inform return policy and inventory decisions.

### 🤖 &nbsp; AI Shopping Assistant &nbsp;`In Progress`

> **Python · LangChain · RAG · Vector DB · LLMs**

A conversational shopping agent that goes far beyond keyword search. It grounds every response in a **RAG pipeline** built over product descriptions and user reviews — eliminating hallucinations and anchoring recommendations in real data.

The system combines **hybrid recommendations** (content-based + collaborative filtering), asks intelligent clarifying questions before surfacing results, and maintains **persistent session memory** across multi-turn dialogues. The end goal: a shopping companion that *understands context*, not just queries.
